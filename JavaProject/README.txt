running the jar file in cmd:
java -cp NetflixTemplate-1.0-SNAPSHOT.jar com.mycompany.netflixtemplate.Main

Jar file is in the target folder



Information about the project:
-Simulation class includes relative paths that are Windows specific (lines 366,379)
-At the start of the program the user has to either start a new simulation or load and resume the loaded simulation
-The simulation has to end in order to load or start a new simulation (if it is not terminated by closing the window it has a time limit of 500 units of time before the end of the simulation
-User number is set to 100 and channel count is set to 10
